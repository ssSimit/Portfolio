TraceCraft

An educational Unity game designed to help players learn and explore geometry through interactive tracing and painting of 2D and 3D forms. The core idea was to make abstract mathematical concepts more engaging by turning them into hands-on, visual interactions.

About the Project

This project was created as an educational game focused on helping players better understand geometric forms through active participation rather than passive instruction. The primary goal was to design an experience where learning happens through interaction, by tracing, rotating, and painting objects instead of simply viewing them. For 2D forms, players trace dotted outlines using a pencil-like mechanic implemented with Unity’s Line Renderer. The same system is extended for painting, allowing players to fill shapes through guided strokes. This approach reinforces shape recognition, hand–eye coordination, and spatial awareness in a way that feels playful and intuitive. For 3D forms, the game introduces rotatable models such as cubes and cylinders. Players can freely rotate these objects and paint them by shooting color onto different faces. Each face can be colored independently, encouraging exploration of three-dimensional structure and helping players understand how 3D objects are composed of multiple surfaces. Overall, the experience was designed to be calm, accessible, and creative, using soft visuals and simple interactions to keep the focus on learning. The project emphasizes clarity, feedback, and experimentation, making it suitable for younger learners while remaining technically robust.
